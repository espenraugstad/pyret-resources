Here is a short video on how you can get Pyret code into a Github repo.

Summary

  1. Create a new reposityory on Github.*
  2. Clone the repository to your local computer.**
  3. Open the folder with the repository in a terminal on your computer.
  4. Go to the Pyret file you have saved, click "File" and then "Download".
  5. Locate the downloaded file and drag it into the folder containing your repository.
  6. In the terminal, type 
    git status
    to check that the file has been registered.
  7. Finally, add the file, commit it, and then push the changes to Github.

Notes:
* It is not necessary to create a new repository if you want to put your code in an existing repo.
** If you have an existing repository that is already on your local computer, this step is not necessary.
